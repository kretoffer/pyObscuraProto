# pyObscuraProto: Python-обертка для C++ библиотеки ObscuraProto

## Введение
`pyObscuraProto` — это Python-обертка, которая обеспечивает беспрепятственный доступ к высокопроизводительной C++ библиотеке `ObscuraProto`. Эта обертка позволяет Python-разработчикам использовать мощь и эффективность базовой C++ реализации.

## Установка

``` bash
    pip install pyObscuraProto
```
